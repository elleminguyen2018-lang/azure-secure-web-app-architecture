# Lessons Learned

## What I learned

- Azure Application Gateway works at Layer 7
- WAF helps protect web apps from common attacks
- Bastion allows secure access without exposing VMs to the internet
- Backend VMs can stay private with no public IPs
- NSG rules are critical for traffic flow
- Health probes determine whether traffic reaches the backend servers

## Reflection

This project helped me understand how Azure services work together in a more realistic and security-focused design instead of just deploying resources individually.
